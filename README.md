# Project Overview

This project presents an exploratory data analysis and interactive dashboard built using Microsoft Power BI to analyse movies available on Amazon Prime Video. The dashboard provides insights into movie distribution, genres, ratings, release trends, and other key attributes to support data-driven understanding of streaming content.

# AIM:
The goal of the project is to demonstrate data visualization, exploratory analysis, and dashboard design skills using real-world entertainment data.

# Objectives

1. Analyse the distribution of movies available on Amazon Prime.
2. Explore trends in movie releases over time.
3. Examine genre popularity and content ratings.
4. Identify patterns in movie duration and production.
5. Build an interactive and user-friendly Power BI dashboard.

# Dataset Description
The dataset used in this project contains information about movies available on Amazon Prime Video, including:
- Movie title
- Release year
- Genre(s)
- Duration
- Content rating
